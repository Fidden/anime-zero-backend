# AnimeZero

Online anime cinema website

## Tech Stack

- Laravel

### Installation

1. Clone repository

```sh
git clone https://github.com/Fidden/anime-zero-recode.git
```

2. Install composer dependencies

```sh
php composer.phar install
```

4. Start local server

```sh
php artisan serve
```

### How to parse content ?

```sh
php artisan parse
```
