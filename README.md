### Установка

1. Клонируй репозиторий

```sh
git clone https://github.com/Fidden/anime-zero-recode.git
```

2. Установи зависимости composer

```sh
php composer.phar install
```

3. Установи NPM пакеты

```sh
npm install
```

4. Собери приложение

```sh
npm run prod
```

5. Запусти сайт

```sh
php artisan serve
```

### Как добавлять контент

```sh
php artisan parse
```
